## 配置

- 自定义 favicon：将 `favicon.[svg|ico|png]` 文件放入 `/static` 目录
- 自定义 logo：将 `logo.svg` 文件放入 `/static` 目录
- 自定义 css：将自定义的 css 文件写入 `/assets/css/custom.css`
